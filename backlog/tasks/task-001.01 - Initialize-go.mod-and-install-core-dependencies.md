## Description

<!-- SECTION:DESCRIPTION:BEGIN -->
Set up the Go module and install all required dependencies for the llxt project.

## Files to Create/Modify
- `go.mod` - Initialize module
- `go.sum` - Generated by go get

## Implementation Steps
1. Initialize go module: `go mod init github.com/takinprofit/llxt`
2. Install core dependencies:
   - `go get github.com/urfave/cli/v3@latest`
   - `go get resty.dev/v3`
   - `go get github.com/jedib0t/go-pretty/v6`
   - `go get github.com/samber/oops`
   - `go get github.com/knadh/koanf/v2`
3. Install koanf providers/parsers:
   - `go get github.com/knadh/koanf/providers/file`
   - `go get github.com/knadh/koanf/providers/confmap`
   - `go get github.com/knadh/koanf/parsers/toml/v2`
4. Run `go mod tidy`

## Commit
```bash
git add go.mod go.sum
git commit -m "chore: initialize go module with dependencies"
```
